Basic

A single alert is rendered using t-alert based of different type, having header and message as input.

Closable

Alert are closable using close icon being displayed on top right corner. In order to disable closable messages, set enableClose to false. Note to hide alert use onCloseClick output which is used to capture close icon event and then alert can be hidden using *ngIf.
