Inject ToastService into component.
Toast types are :-
success, info, error, warning
Toast positions are :-
toast-top-right, toast-top-left, toast-bottom-right, toast-bottom-left, toast-top-center, toast-bottom-center, toast-top-full-width, toast-bottom-full-width
ActiveToast provides following options :-
toastId: number (Your Toast ID. Use this to close it individually)
message: string (message of your toast. Stored to prevent duplicates)
portal: ComponentRef<any> (reference to the component)
toastRef: ToastRef<any> (a reference to your toast)
onShown: Observable<any> (triggered when toast is active)
onHidden: Observable<any> (triggered when toast is destroyed)
onTap: Observable<any> (triggered on toast click)
onAction: Observable<any> (available for your use in custom toast)
showToaster() {
